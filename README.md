# Task 5: Research Report on Social Engineering Attacks A behavioral analysis of the human element in cybersecurity, detailing how attackers bypass technical boundaries using psychological manipulation. 
Focus Areas: The social engineering lifecycle, Phishing (Spear Phishing, Whaling, Clone Phishing), Pretexting, and USB/Digital Baiting.
Highlights: Examines massive breaches like the 2020 Twitter VIP Hijacking and the MGM Resorts Ransomware attack, while outlining prevention strategies like Phish-Resistant MFA (FIDO2) and Two-Person Verification processes.
